The enclosure consists of two parts:

- Bottom Case
- Lid

The design provides sufficient clearance for the ESP32 board while maintaining a compact and printable structure.

| Parameter | Value |
|-----------|------:|
| ESP32 board | 55 × 28 mm |
| Clearance | 1 mm |
| Internal cavity | 57 × 30 mm |
| Wall thickness | 2 mm |
| Bottom thickness | 2 mm |
| Wall height | 14 mm |
| Lid thickness | 2 mm |
| Outer dimensions | 61 × 34 mm |

---

## Features

- Separate bottom case and lid
- USB cable opening
- Four mounting holes
- Rounded corners (Fillet)
- Designed for 3D printing
