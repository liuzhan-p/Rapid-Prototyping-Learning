
This project is a simple mounting plate designed in Autodesk Fusion 360 as part of my Rapid Prototyping learning journey.

The objective was to practice fundamental CAD operations that are commonly used in hardware prototyping.

| Item | Value |
|------|------|
| Length | 100 mm |
| Width | 60 mm |
| Thickness | 5 mm |
| Mounting Holes | 4 |
| Hole Diameter | 3.2 mm (M3 clearance) |
