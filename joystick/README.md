# 🕹️ Fusion 360 Joystick Prototype

A simple joystick prototype designed in **Autodesk Fusion 360** to demonstrate CAD modeling, mechanical assembly, and motion simulation. This project was created as a personal practice project to improve product design and prototyping skills.

---

## Features

- Parametric CAD model
- Mechanical assembly
- Motion simulation
- Simple and clean design
- 3D-printable components


## Design

The joystick consists of four main components:

- Base
- Handle
- Shaft
- Knob

The handle is assembled using Fusion 360 joints, allowing realistic movement through motion simulation.
